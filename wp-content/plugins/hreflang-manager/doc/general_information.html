Introduction

The Hreflang Manager plugin provides you with an easy and reliable method to implement hreflang in WordPress. It comes with a large number of customization options, a dedicated menu to bulk import hreflang data, a feature to automatically sync the websites of your network, the ability to import and export the plugin data, and more.

Quick Start

  1. Install the plugin on your website with the instructions included in the Installation section of the plugin manual.
  2. Set up hreflang on one website of the network with the instructions provided in the Introduction and Tutorials sections of the manual.
  3. Repeat the hreflang implementation on the other websites of the network or automatically sync all the websites of the network with the instructions provided in the Sync section of the manual.
